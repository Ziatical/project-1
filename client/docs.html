Homepage

GET

/api/findCountry GET HEAD

Query Params

currency - A currency of choice to filter by.

Returns

A JSON array of countries with that currency. Format: [string]

Example

Request: /api/findCountry

Result: ["country1", "country2", "country3"]

/api/findCapital GET HEAD

Query Params

country - A country to find the capital for.

Returns

A JSON array of capitals within that country. Format: [string]

Example

Request: /api/findCapital

Result: ["capital1", "capital2"]

/api/findCountries GET HEAD

Query Params

region - A region to find countries from.

subregion (optional) - A subregion to find countries from and filter (Ex. Southern, Northern, etc.).

Returns

A JSON array of countries from within that region. Format: [string]

Example

Request: /api/findCountries

Result: ["country1", "country2", "country3"]

/api/findSymbol GET HEAD

Query Params

currency - A currency to get the symbol for.

Returns

A single string. Format: string

Example

Request: /api/findSymbol

Result: symbol

POST

/api/removeCountry POST

Query Params

country - A country to remove.

Returns

A single string stating the status. Format: string

Example

Request: /api/removeCountry

Result: Country was successfully removed.

/api/changeCurrency POST

Query Params

country - The country to change currency of.

currency - The new currency to be changed into.

Returns

A single string stating the status. Format: string

Example

Request: /api/changeCurrency

Result: Country successfully changed to new currency.
